LAATIKKOMALLI Elementit ovat laatikoita, joilla on kehys ja ulkopuolelle jäävä marginaali.

Tuotteet

Kehys = Border - Oletuksena näkymätön. h2 { border: 1px solid #000000; } Täyte = Padding - Kehyksen ja sisällön välissä. - Joillakin elementeilla on oletustäyte. Marginaali = Margin - Kahden vierekkäisen elementin kehysten väliin jäävä alue. - Joillain elementeillä on oletusmarginaali Oletusmarginaalit ja -täytteet voidaan poistaa css-säännöllä: * { margin: 0; padding: 0; }
